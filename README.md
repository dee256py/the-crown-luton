# The Crown Luton

A full-stack pub website built to manage events, bookings and performer applications.

## Features

* View upcoming events
* Book private events
* Apply to perform at the pub
* Admin booking management

## Tech Stack

### Backend

* Node.js
* Express
* SQLite

### Frontend

* React
* Tailwind CSS
